# Tekpaint

Paint like in C with LibLapin Library (SMFL overlay)

    Dev period: 10 january 2016 - 26 january 2016

Functionalities
=============
* Brush
* Brush size
* Square brush
* Line
* Pain bucket
* Pipette
* Color palette
* Rubber
* Filters
    * Red
    * Blue
    * Black & White
* Save
    * BMP
        * 24 bits
        * 32 bits
    * FTN (Own format)
